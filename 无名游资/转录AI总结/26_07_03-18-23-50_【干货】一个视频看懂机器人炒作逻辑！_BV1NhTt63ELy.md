### 3. 机器人板块的核心分析框架 ⭐⭐⭐

UP 主认为影响机器人涨跌的核心因素有两个：

**因素 A：量产节奏（资本开支逻辑）**
- 盯紧机器人生产商的订单和采购数据
- 零部件卖得越多 → 业绩越好 → 股价水涨船高
- ⚠️ 投资人的特点是"抢跑"——闻到潜在订单的气味就开始涨，等业绩真正兑现股价可能已经见顶

**因素 B：终端销量（产品卖得好不好）**
- 卖得好 → 良性循环：销量大 → 零部件需求大 → 全产业链受益
- 卖得不好 → 库存积压 → 零部件拿不到增量订单 → 业绩无法兑现 → 股价回调
- 参考案例：之前的新能源板块——涨得多高跌得多惨

### 4. 资本开支作为"替代观察指标"

- 如果短期销量不行，市场会把关注点从"业绩"切换到"资本开支"
- 资本开支 = 厂商在建厂、买设备、铺产线上的投入，**代表远期预期**
- 只要厂商还在持续投钱扩产，说明他们对未来有信心，逻辑还在
- 但资本开支逻辑的强度比业绩弱很多——市场不喜欢等待
