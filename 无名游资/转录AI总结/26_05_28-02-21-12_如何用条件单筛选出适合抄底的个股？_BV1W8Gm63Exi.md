# 如何用条件单筛选出适合抄底的个股？

## 核心立场

UP 主分享了一套**"卖盘枯竭"抄底系统**——通过条件选股公式筛选出量能萎缩到极致的个股，然后在关键支撑位买入，跌破就止损。核心逻辑是**用极小的亏损去博弈可能的波段反弹，靠盈亏比而非胜率赚钱。**

## 金融与交易知识点

### 1. 卖盘枯竭选股公式 ⭐⭐⭐

UP 主在"同花顺动态分组"中设置以下条件：

| 条件 | 参数 | 目的 |
|:----:|:----:|:----:|
| 总市值 | 800亿 ~ 8000亿 | 排除小盘股（防止被操控） |
| 营收增长率 | >1% | 基本面有支撑 |
| **量比** | **0.1 ~ 1.0** | ⭐ 核心指标：成交量极度萎缩 |
| 今日涨跌幅 | -1% ~ +2% | 不能太大涨也不能太大跌 |

- 每天收盘后刷新，从 5000 多只股票中选出约 20 只
- 不是所有入选的都适合买入——**还需要人工判断是否在关键支撑位**

### 2. 卖盘枯竭的核心理念 ⭐⭐

- **成交额降到历史最低量附近** → 这意味着该卖的人已经卖得差不多了
- 卖盘枯竭后 → 如果有资金来拉升 → 会比较容易（因为抛压小了）
- 但也可能继续跌 → 因为"没有买盘"也是问题
- ⭐ **这不是高胜率策略，而是高盈亏比策略**

> "交易十次里对四次都可以盈利，错六次都没关系，因为盈亏比非常可以。"

### 3. 实际案例演示

UP 主展示了三笔实际操作：

1. **平安银行** ✅：在量比极低、接近支撑位时买入 → 成功抄底
2. **申万宏源** ✅：买入后上涨 → 但又被跌回来了，目前仍盈利
3. **宝钢股份** ❌：跌破支撑位 → 止损出局

### 4. 如何从选股结果中挑选

并不是选出来的所有票都买——需要人工筛选：

- **必须是在关键支撑位附近**（前低/筹码密集区）
- 成交额要在**历史最低量附近**
- **趋势必须是上升趋势**或**至少没有走坏**
- 上升趋势的特征：高点不断创新高 + 低点不断创新高

**举例分析：**
- **牧原股份**：成交额最低时在 10 亿左右 → 到 10 亿附近可考虑抄底
- **爱尔眼科**：已跌到历史大底（历史最低点位）→ 跌 2% 止损，涨 10%+ 预期
- **中国核电**：有主力建仓痕迹，最低量 6.38 亿，当前 9.29 亿 → 还不太够低
- **宁波银行**：上升趋势中回调到支撑位 → 可博弈
- **美的集团**：入选但不在支撑位 → 不符合买点

### 5. 止损纪律 ⭐

- 跌到支撑位买入 → 跌破支撑位**立即止损**
- 亏损预期：约 -2%（小亏损）
- 收益预期：约 +10%（中波段）
- 每一次失败亏损都很小 → 但只要成功一次就够覆盖多次亏损

### 6. 波段抄底的升级版

- 如果第一次抄底后跌破支撑位 → 止损
- 寻找下一个支撑位 → 再抄底 → 再设止损
- 即使连续错几次，总亏损都不会超过 5%
- 但如果对一次，一次盈利就是两位数的

### 7. 药明康德的再买入计划

UP 主表示还会再回来买药明康德：
- 等成交额跌到历史低位（约 20 亿）→ 且价格跌到支撑位（约 8 元）→ 重新买入

## 其他

- 此方法适用于**大市值（800亿+）** 个股——小市值股量的量价关系容易被操控
- 操作周期偏波段（数天到数周），不是超短
- ⭐ 方法论价值极高——量比 0.1-1.0 是核心筛选指标
