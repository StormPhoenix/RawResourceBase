## 内容摘要：系统化投资知识体系

### 一、选股与资金判断

**不同周期的选股策略：**
- **短线**：题材 + 资金 + 图形 + 量能 → 最强板块的龙头前排
- **中线**：景气行业 + 基本面 + 估值 → 高 ROE/高毛利/现金流好
- **长线**：护城河 + 管理层 + 长期逻辑 → 核心壁垒行业龙头

**资金认可的判断标准：**
- 走势强势（涨停连板、分时抗跌、回踩关键位不破）
- 量能健康（放量上涨、缩量回调）
- 龙虎榜机构/北向资金/板块联动佐证

### 二、打板、连板与妖股

**封板质量判断：**
- 早盘硬封/缩量封板 = 强
- 尾盘烂板/反复开板 = 弱
- 次日高开 5% + 竞价量足 = 强；低开/平开/缩量 = 弱

**妖股特征识别：**
- 流通市值 50 亿 + 股价 10 元 + 长期低位横盘
- 连续涨停 + 高换手率 + 无视大盘
- 传统行业/业绩差/无机构持仓
- 澄清公告反而涨 → 资金接力驱动

### 三、止损止盈规则

- **短线**：-5% 硬止损
- **中线**：-10% 止损
- **长线**：-20% 最大回撤
- 止损条件：买入逻辑消失、题材退潮、业绩爆雷、技术破位

### 四、做 T 操作

- **正 T**：低开回踩支撑位买入 → 冲高卖出
- **反 T**：高开冲高回落时卖出 → 回落支撑位买回
- 禁忌：单边大跌不做 T、频繁做 T 亏手续费

### 五、情绪判断与技术分析

**分歧→主升的量价特征：**
- 分歧阶段：回调解禁阴小阳 + 缩量 + 不破 5 日线
- 主升启动：放量涨停 + 阳包阴 + 突破前高

**量价四象限：**
| 价格 | 成交量 | 含义 |
|:---:|:-----:|:----:|
| 涨 | 增 | 多头排列，健康 |
| 涨 | 缩 | 顶背离，见顶风险 |
| 跌 | 增 | 死叉，下跌加速 |
| 跌 | 缩 | 底背离，见底信号 |

### 六、实战战法全流程

1. **盘前（8:30-9:15）**：查隔夜消息、外盘、板块涨幅
2. **竞价（9:15-9:25）**：关注涨停家数、高开幅度、竞价量能
3. **开盘（9:30-10:00）**：观察分时承接、量能、板块联动
4. **持有阶段**：盯 5 日线、分时支撑
5. **卖出阶段**：-5% 止损 / +10%~20% 分批止盈

### 七、基本面与估值

- 短线：无需估值，情绪+资金驱动
- 中线：PE 10-20 倍为合理，结合行业增速
- 长线：DCF 现金流折现 + ROE + 自由现金流
- **警惕假利好**：无官方公告 + 内容模糊 + 小作文传播 = 假利好

### 八、稳健持仓规划模板

- 持仓 3-5 只标的，不重仓单一板块
- 中线 50%（3-12 个月，-10% 止损）
- 短线 20%（快进快出，-5% 硬止损）
- 长线 30%（1 年以上，-20% 回撤底线）
