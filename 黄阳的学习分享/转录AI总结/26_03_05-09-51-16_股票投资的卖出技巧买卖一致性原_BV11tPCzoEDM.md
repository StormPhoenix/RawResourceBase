# 股票投资的卖出技巧，买卖一致性原则

## 核心立场

UP 主分享一个简单实用的卖出方法论——**买卖一致性原则**。一句话概括："你以什么理由买入，就以对应的理由来决定是否卖出。"这个方法能解决"会买的是徒弟、会卖的是师傅"的千古难题，让卖出决策变成本能反应。

## 金融与交易知识点

### 1. 买卖一致性原则：原理（核心 ⭐⭐⭐）

> "买卖一致性的意思就是字面的意思——你买股票肯定有买入的理由，任何理由都可以。买卖一致性的意思就是你卖出的时候你要审视一下你买入的理由，如果你觉得你买入的理由都不成立了，那你就可以卖出了。"

**核心理念**：
- 买入时的理由 = 卖出时的标准
- 不需要看消息、看情绪、看大盘涨跌
- 只需检查当初的买入理由是否还成立

**实操建议**：
- 购买新公司时，最好用笔记下买入理由清单
- 定期复盘对照清单
- 一旦大部分理由不成立 → 考虑卖出

### 2. 案例一：某头部白酒公司（价值投资场景 ⭐⭐）

**三个买入理由**：
1. 核心产品送领导最体面（生活常识可观察）
2. 核心产品供不应求，市场价 > 零售价（官方 app 上还要抢）
3. 当前估值不贵，PE ~20（历史均值 32，低于 32 都可以接受）

**卖出标准**：不复盘消息、情绪、大盘，只看这三点是否还成立
- 三条都成立 → 继续持有
- 理由不成立 → 才考虑卖出

### 3. 案例二：某四大行投资（低估修复场景 ⭐⭐）

**买入理由**：只有一个——"太便宜了"
- 当年估值便宜到令 UP 主不敢相信
- 算过账：只要公司不倒闭，业绩稳住哪怕略微下滑，这个价格买都是赚的
- 经营水平都不考虑了

**卖出标准**：唯一标准——"不再便宜了"
- 用市盈率判断：低于均值一个标准差（绿线）买入
- 高于均值一个标准差（红线）卖出
- 股价大涨时控制在估值区间内操作，既完整吃到从低估到高估的整段收益，又避免贪婪

> "当年很多人看出来这家公司低估，但卖出的标准不清晰。大多数人涨个 30% 就赶紧跑了，结果错过了一个低风险能翻几倍的机会。"

### 4. 案例三：技术分析场景（投机场景 ⭐）

- 画好支撑位和压力位
- 跌到支撑位买入
- 涨到压力位卖出
- 思路清晰，钱赚得明白

### 5. 听别人推荐时的用法（⭐）

- 听 X 的建议买入 → 参考 X 的建议卖出
- 这也算买卖一致性原则

### 6. 核心价值：便于复盘（核心 ⭐⭐）

> "这个方法的**最大好处就是便于复盘**。如果你的一笔投资做亏了，找原因也很简单——就是把当初买入的理由翻出来再审一遍，看看这理由可不可靠。"

- 听消息追热点亏了 → 就知道这理由不可靠
- 坚持复盘几年 → 自然知道什么样的理由才是有效的买入逻辑
- 投资能力练出来了，稳定盈利就不难了
