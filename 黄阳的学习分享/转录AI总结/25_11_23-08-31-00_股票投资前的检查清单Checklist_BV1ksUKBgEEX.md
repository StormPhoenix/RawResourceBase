# 股票投资前的检查清单 Checklist

## 核心立场

UP 主在大盘暴跌日（跌 2.45%）推出这套投资检查清单，帮助投资者**用系统化方法替代随意操作**。核心主张：买股票前应该比买车买手机做更多研究，"你买个手机壳都要货比三家，怎么换到买股票就随便下单了？" 清单分为四部分（企业基本情况→行业分析→财务指标→股价表现），按顺序过一遍，保证不踩大坑。

## 金融与交易知识点

### 1. 为什么需要检查清单（核心 ⭐⭐）

**不熟悉公司的结果**（与之前课程呼应）：
- 股价涨：拿不住 → 赚十几个点就跑 → 错过翻倍
- 股价跌：不愿割肉 → 越亏越大
- **根本原因：没有公司的内在价值作为标尺**

**沉没成本不参与重大决策**：
> "账户上的浮亏是你过去的错误造成的，这已经是沉没成本了。当前要考虑的是：假如你现在是空仓，让你重新审视这家公司，你是否会去投资它？如果答案为否，那应该怎么做就不用我明说了。"

### 2. 第一部分：企业基本情况（⭐）

**核心内容**：
1. 公司全称（了解全称是最基本的礼貌）
2. 公司简介 + 使命与愿景（最快了解公司的方法）
3. 核心产品和服务（官网和财报中都有）
4. 产品是否有定价权（填是/否/不知道，后面慢慢研究）

**使命与愿景的实战用法**——比亚迪 vs 赛力斯对比：
- 比亚迪使命："用技术创新满足人们对美好生活的向往" → 强**技术**导向 → 产品堆技术
- 赛力斯使命："用户至上、安全是生命" → 强**用户**导向 → 产品注重体验
- "通过使命对比，这两个车企的区别一下就搞清楚了"

### 3. 第二部分：行业分析（⭐）

**需要考察的点**：
1. 所属行业 + 行业生命周期（导入期/成长期/成熟期/衰退期）
2. 竞争格局（百舸争流/一超多强/巨头垄断）
3. 产业链位置——微笑曲线（尽量选两端，避开中间制造环节）
4. 上下游议价能力
5. 股权结构（国企/民企/家族/合伙/资本控股）

### 4. 第三部分：财务指标（核心 ⭐⭐）

**业绩趋势图**：
- 将营收数据转为柱状图 → 一眼看出成长性
- 格力案例：8 年业绩 1800 亿零增长 → 一目了然

**资产负责表**：
- 将数据转为图表观察
- 比亚迪：应付账款高 → 强势甲方

**ROE（净资产收益率）**：
- 效率指标，用于横向对比为主
- 同类型公司选 ROE 高的

**分红**：
- 看分红比例和稳定性
- 排掉"掏空家底式分红"的风险

### 5. 第四部分：股价表现（核心 ⭐⭐）

**市值规模**（横向对比法）：
> "把一个小米卖了可以换六个吉利——虽然小米还有手机业务，但跟吉利相比这个差异是不大的。要么小米市值高估了，要么吉利低估了。"

**估值指标**：
- PE（市盈率）→ 适用于利润稳定的公司
- PS（市销率）→ 适用于利润不稳定的成长型公司
- 把历史数据做成趋势图 + 计算均值 ± 标准差

**K 线**：判断短期趋势（上升/下降/横盘）

**筹码分布**（实用技巧）：
- 获利比例太高 → 买进容易被套
- 获利比例低 + 基本面稳定 → 下跌概率小 → **适合加仓**
- "我自己一般会这样搞，胜率还比较高"

### 6. 核心思想（重要 ⭐⭐）

> "填不了 = 你不了解公司业务 → 直接不买就行，起码不会亏，这就保住本金了。"

- 检查清单的作用是**底线思维**——不能保证赚钱，但一定不会踩大坑
- 不是一次性工作——需要定期拿出重新审视
- 现在有 AI 工具辅助："自己分析不了的可以让 AI 先分析，把结论填上，后面自己慢慢验证"
