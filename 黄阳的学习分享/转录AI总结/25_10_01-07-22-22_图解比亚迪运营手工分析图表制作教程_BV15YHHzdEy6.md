# 图解比亚迪运营，手工分析图表制作教程

## 核心立场

UP 主手把手教投资者如何用 Excel 制作公司分析图表。核心方法论：**用数据可视化代替感觉判断**，通过 5 组核心图表（业绩趋势、业绩构成、利润与现金流、资产结构、市销率估值）快速了解公司状况，筛选出真正值得深入研究的公司。

## 金融与交易知识点

### 1. 业绩趋势图（核心 ⭐⭐）

**两个核心图表**：

**① 营收 + 主业现金流组合图**
- 选择"销售商品服务收到的现金"（主业现金流）而非总现金流
- 营收和主业现金流叠在一起看 → 判断回款质量
- 如果两者长期背离 → 公司可能有问题

**② 净利润 + 现金流金额组合图**
- 比亚迪案例：现金流金额**远高于净利润**
- 三种可能原因：① 大量非现金费用（折旧）② 经营性负债增加③ 存货和应收账款
- **现金流 > 利润 = 加分项**

### 2. 业绩构成分析（⭐）

**两个维度**：产品维度和市场维度

**比亚迪案例分析**：
- 汽车 + 手机代工（大家不知道比亚迪手机代工也很强）
- 境外业务占比约 30%，增速比国内还好些
- "目前增速主要靠国外市场撑着"

### 3. 成本与费用分析（⭐）

- 毛利率分析：比亚迪维持在 20% 左右，行业中较高
- 2021-2022 毛利率下滑→锂矿价格飞涨→后面恢复
- 销售费用和管理费用控制好（约 2.5%-3%）
- 研发费用占比 7.5%→超过净利润率 5%→"赚到的钱都拿来搞研发了"

### 4. 资产结构分析（重要 ⭐⭐）

**比亚迪的应付账款问题**：
- 应付账款极高
- 流动现金+应收款 < 应付款 → **存在一定风险**
- 好的一面：欠供应商的钱相当于无息借款
- 坏的一面：容易降低供应商积极性
- 趋势在改善——"以后每发财报的时候要关注这个数据是否持续改善"

### 5. 市销率估值（核心 ⭐⭐）

**为什么用 PS 不用 PE**：
- 比亚迪利润不稳定（研发投入比利润还高）
- 成长型公司看市销率（PS = 市值/总营收）

**标准差参考线**：
- 计算 PS 历史数据的均值和标准差
- 黄线 = 均值、上线 = 均值+1σ(高估)、下线 = 均值-1σ(低估)
- 当前比亚迪 PS 处于**相对较低**位置

### 6. 手工数据制作五步法（实操 ⭐）

1. **下载原始数据**（同花顺网站）→ 财务指标下载
2. **转为滚动数据**（累计数据→4个季度滚动求和）
3. **计算市值**（不复权股价 × 总股本，每周取一次）
4. **制作组合图**（Excel 组合图功能，把业绩和市值叠在一起）
5. **计算 PS/PE + 标准差**（均值±σ作为参考线）

**时间建议**：熟练后约 1 小时完成一套图表

## 其他

- 核心概念：滚动数据、组合图、市销率、标准差
- 数据来源：同花顺网站（免费下载）
- 工具：Excel（基础图表功能即可）
- 关键公式：滚动净利润 = 本期累计 + 去年下半年 = 去年年报 - 去年半年报 + 今年半年报
- 核心原则：**"凡是没业绩增长性的公司就不具备投资价值，坚守这一原则基本不会踩大坑"**
