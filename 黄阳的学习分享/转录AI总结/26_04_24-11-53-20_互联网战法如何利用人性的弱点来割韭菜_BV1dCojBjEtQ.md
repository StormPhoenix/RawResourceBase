# 互联网战法如何利用人性的弱点来割韭菜

## 核心立场

UP 主从行为经济学角度，系统性拆解游资和机构利用"**有限注意力偏差**"割散户韭菜的套路。本质是：利用人类进化遗留的生理机制（注意力高度集中单一信息源）在现代股市中的错配效应，通过人为提高垃圾股的曝光量吸引散户追涨接盘。

## 金融与交易知识点

### 1. 有限注意力偏差：进化遗留的生理机制（核心 ⭐⭐⭐）

**定义**：我们的大脑一次只能聚焦少数关键信息，自动忽略无关内容

**进化起源**（关键 insight）：
- 远古时期，草丛一动或听到异响 → 大脑瞬间锁定那个异常点 → 其他声音、气味全部屏蔽
- 这是生存优势——如果大脑同时处理海量信息反应迟钝，遇到老虎就来不及跑
- **"这是进化论、生存筛选出来的生理机制"**

**现代社会的错配**（基因与环境的错配）：
- 开车不能看手机（大脑很难同时处理两件重要的事）
- 看电视剧入神时听不到旁边人喊你
- 股市也是典型的现代社会产物 → 凭本能行动容易犯错

### 2. 注意力驱动溢价：关注度如何影响股价（核心 ⭐⭐）

**机制链条**：

> 外部事件 → 某股票关注度瞬间提高 → 潜在买家增加 → 供需失衡 → 股价上涨

**经典案例——金龙鱼 vs 石油**：
- 中东打仗封锁海峡 → 国际油价大涨 → 满屏新闻都是"油"
- A 股开盘：**金龙鱼（食用油）股价最高涨 14%**
- 金龙鱼是做食用油的，跟石油**没有半毛钱关系**
- 但因为名字带"油" → 关注度被拉高 → 股价被推上去

**更多谐音梗案例**：
- 川普竞选领先 → A 股**川大智胜**暴涨
- 拜登选票追上 → 另一家公司**乔治白**大涨（Joe Biden → 乔治白）
- 奥巴马竞选 → **奥柯玛**涨停

> "这些看上去很奇葩的事情发生的本质原因，就是这个有限注意力偏差导致的。"

### 3. 互联网战法的操作流程（核心 ⭐⭐⭐）

**传统模式**：借外部事件驱动（热点布局），把股票拉高卖给散户

**新模式（互联网战法）**：
1. 游资/机构提前低价埋伏基本面极差的垃圾股
2. 安排合作的大 V/流量号造势曝光
3. "暗示或明示大家自己买了某只股票"（甚至不用吹票）
4. 背后的产业链给作品投流（"明明没有关注，但每天还是能刷到他们"）
5. 散户因有限注意力偏差 → 下意识关注并追高
6. 股价拉到高位 → 游资把筹码甩给散户 → 完成收割

**UP 主点评**：
> "越是能力强的人干坏事，他破坏力往往就更大。"

### 4. 散户的根本误区：与虎谋皮（核心 ⭐⭐）

> "很多散户自始至终都没有搞明白一点——那些游资机构在这些垃圾公司上搞这种拉高出货快进快出的炒股方式，这些人到底赚的是谁的钱呢？那肯定赚的不是公司经营成长的钱吧——那必然赚的就是散户亏掉的钱。"

**常见错误心态**：
- "我只要早期先追进去，在游资撤退前提前出来不就行了吗？"
- 相当于问"明知是庞氏骗局，早期参与能不能赚钱"

**UP 主的反驳**：
- 赚钱的前提是"运气足够好"
- 但尝到甜头后就会贪婪 → 不会走
- 即使这次运气好跑了，能保证每次运气都好吗？
- **"股票投资的亏损是按百分比计算的，无论前面已经多少次，只要被套一次，账户大概率就会一夜回到解放前。"**

### 5. 投资最忌讳扎堆（重要 ⭐）

> "投资是很忌讳扎堆的——人多的地方必然会有额外的泡沫和溢价。"

- 用房地产案例类比：2018 年最热的时候跑去买房
- "网上各种大 V 教你怎么买房的攻略，但你兄弟会反驳我说，说说油脂炒热点怎么了，我们起码跟着油脂走还能吃到肉啊"
- "这就是典型的人家把你卖了还帮人家数钱的行为"

### 6. 投资需要对抗本能（重要 ⭐）

**核心启示**：
- 金融市场的运行法则和我们的**远古本能是冲突的**
- 我们的大脑还是一套"适应丛林、不适合现代金融市场"的配置
- 只有通过学习和训练，才能识别并压制这些本能反应
