# 中级课程学习回顾，案例串讲

## 核心立场

UP 主用公牛集团作为案例，把中级课程的所有方法论一次性串讲一遍。**从选股→业务分析→财务分析→估值→K线技术→卖出技巧，形成一个完整的投资闭环。** 核心原则：**公司选对了，长期获利的概率直接高达80%。**

## 金融与交易知识点

### 1. 选股三要点（核心 ⭐⭐）

**① 先了解自己的风险偏好**
- 风险偏好低→选成熟行业大公司
- 风险偏好高→选创新行业成长型公司
- 不匹配→"持股过程特别煎熬，遇到风吹草动就卖"

**② 产品能接触到**
- "你能接触到产品→可以通过产品来分析核心竞争力和定价权"
- 日常生活能接触到的上市公司上千家，完全够用

**③ 业绩好**
- "开公司就是为了赚钱"
- 把十年业绩做成趋势图→一眼就能排除垃圾公司

### 2. 公牛集团案例分析（核心 ⭐⭐）

**业务分析**：
- 电连接产品（插座）→76亿，同增4%
- 智能电工照明→同增
- 新能源充电桩→7.7亿，翻倍增长→"未来业绩增长点"
- 战略出海：愿景调整（"国际民用电工行业领导者"）→关注海外业绩

**财务亮点**：
- 现金+理财142亿，几乎没有应收账款
- 零负债公司（短期借款很少、长期借款没有）
- 分红比例70%（大方，但正常30%）
- 注意：现金流比净利润少→需提高估值要求

**估值**：
- AI计算合理估值1000亿
- 当前市值850亿→在合理范围内
- PE处于历史低位（但上市仅5年，历史数据不够多）

### 3. 卖出技巧：买卖一致性原则（核心 ⭐⭐）

> "你以什么理由买入，就以对应的理由来决定是否卖出。"

**四种场景**：
1. 因业绩增长买入→增速低于目标就卖
2. 因高分红买入→股息率降低就卖
3. 因定价权买入→定价权丧失就卖
4. 听别人建议买入→听别人建议卖出

**特斯拉案例**：因25万纯电车无敌买入→小米汽车抢夺定价权→考虑卖出

### 4. 课程总结（重要 ⭐）

**中级课程核心**：
- 先分析公司→再分析市场→做投资组合→长期配置
- **P（建立股票池）→D（建仓）→C（定期审查）→A（动态调整）**

**高级课程预告**：行为经济学、信息化、宏观周期
- 能提高投资上限但见效慢
- "新手先巩固中级课程，能稳定盈利后再学高级"
